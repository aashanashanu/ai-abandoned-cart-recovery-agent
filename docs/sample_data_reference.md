## Derived: cart_state

The Event Ingest Lambda creates `cart_state` documents automatically when
processing `cart_events`. Each cart gets an entry with:

| Field | Value |
|-------|-------|
| `status` | `active` |
| `check_at` | Event timestamp + 30 min |
| `cart_value` | From the cart event |
| `currency`, `device_type`, `session_id` | Copied from cart event |

The workflow picks these up when `check_at < now`.
